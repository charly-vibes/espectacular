# espectacular

**espectacular** is a behavioral verification layer for the charly AI development ecosystem — it lets you write machine-readable specs that describe what your tools should do, then continuously check that behavior with `ah check`. Think of it as a contract test harness for AI-assisted workflows: specs declare the expected shape of CLI output, adapters run the actual commands, and the gate rejects merges when behavior drifts.

- [Command Reference](commands.md) — full reference for `ah check`, `ah explain`, `ah doctor`, `ah signals`, and more
